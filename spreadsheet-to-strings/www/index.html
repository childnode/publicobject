Spreadsheet to strings: a localization tool for Android

How it works

  1. Create a new Google spreadsheet to manage your translation text. Each row in the spreadsheet is a string to be translated. Each column is a language that you support.
  2. Get your worldly friends to help out by using the collaboration features in Google Docs.
  3. This app will generate strings.xml resource files for your Android project from the spreadsheet.
